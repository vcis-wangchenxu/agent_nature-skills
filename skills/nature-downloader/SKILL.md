# Legal paper acquisition and organization

## Workflow

1. Prefer official open-access sources: publisher OA page, arXiv, OpenReview, ACL Anthology, PMLR, NeurIPS/ICML/ICLR proceedings, author pages and official GitHub repositories.
2. Collect metadata: title, authors, venue, year, DOI/arXiv/OpenReview URL, code URL, dataset URL.
3. Organize files into a consistent folder structure:
   - `papers/`
   - `bib/`
   - `code-links.md`
   - `benchmarks/`
   - `notes/`
4. For inaccessible papers, suggest legal routes: institutional library, interlibrary loan, author request, accepted manuscript.

## Red lines

Do not bypass authentication, scrape restricted content, share pirated PDFs, use stolen credentials or instruct paywall circumvention.
