# Shared rules for agent_nature-skills

这些文件不是独立 skill，而是所有 `nature-*` skill 的公共判断层。

- `field-taxonomy.md`：RL/MARL/LLM/LLM Agent 论文类型、贡献类型与常见术语。
- `reproducibility-checklist.md`：代码、实验、prompt、模型版本、随机种子、算力和环境复现清单。
- `nature-readiness-rubric.md`：投稿前 readiness 评分表，用于自审、审稿模拟和回复规划。

所有技能应优先遵守这些公共规则，再执行自身任务。
